# Fine-Tuning

A collection of fine-tuning experiments for various language models.

## Models

- **Llama-3.2-3B-Instruct** - Fine-tuning experiments with Meta's Llama 3.2 3B model
- **Phi-3-mini-4k-instruct** - Fine-tuning with Microsoft's Phi-3 mini model using 4-bit quantization

## Structure

Each model directory contains:

- Jupyter notebooks with training code
- Dataset files
- Model configurations

## Getting Started

Navigate to the specific model directory and follow the instructions in the respective README or notebook.

## Requirements

- Python 3.8+
- PyTorch
- Transformers
- Jupyter Notebook
